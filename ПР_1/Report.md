## Введение

### Цель работы

Целью данной работы является исследование возможностей платформы Apache Spark для обработки и анализа больших данных на примере журнала обработки заявок (process mining). В ходе работы необходимо загрузить данные в HDFS, выполнить их предварительную обработку, провести аналитику с использованием Spark SQL и визуализировать полученные результаты для выявления "узких мест" бизнес-процесса.

### Постановка бизнес-задачи

Предметная область — анализ бизнес-процессов на основе логов обработки заявок/документов. Данная работа имитирует реальный процесс рассмотрения заявок (например, кредитных или иных документов) с последовательными этапами, временными метками, исполнителями и статусами выполнения.

**Бизнес-задачи:**
1. Загрузить и обработать журнал обработки заявок в HDFS
2. Подготовить данные для анализа: очистка, преобразование временных меток, вычисление длительности этапов
3. Выявить узкие места процесса — этапы с наибольшей средней длительностью
4. Обнаружить аномально долгие этапы с использованием метода 3-сигм (Z-score > 2)
5. Визуализировать результаты с помощью гистограмм и ящиков с усами (boxplot)
6. Сформулировать рекомендации по оптимизации бизнес-процесса

### Описание данных

В работе используется синтетический датасет, сгенерированный специально для данного исследования. Параметры генерации:
- **Количество заявок:** 2 000 000
- **Количество записей (этапов):** ~10 000 000
- **Объем файла:** ~1.2 ГБ
- **Формат:** CSV

**Структура данных:**

| Поле | Тип | Описание |
|------|-----|----------|
| request_id | integer | Уникальный идентификатор заявки |
| stage | string | Название этапа обработки |
| start_time | timestamp | Время начала этапа |
| end_time | timestamp | Время окончания этапа |
| status | string | Статус завершения этапа |
| operator | string | Исполнитель |
| doc_type | string | Тип документа/заявки |


В данные искусственно внесены аномалии (2% заявок имеют аномально долгие этапы согласования и валидации), что позволяет протестировать методы обнаружения выбросов.
